## Chú ý chung

1. Khi Mod, vui lòng giải nén ra, rồi mod đủ file, không được bỏ 1 file nào để tránh lỗi!

2. Sử dụng skin mặc định để có trải nghiệm tốt nhất.

## Chú ý nếu bạn mod bằng IOS

* Mod Tay

1. Cóp file trong từng folder của file mod vào từng folder của folder game, NGHIÊM CẤM copy folder Resources gắn trực tiếp vào folder game gốc.

2. Vui lòng để ý bạn đã cóp đủ file hay chưa, tránh trường hợp lỗi game đổ lỗi cho tớ =)

* Mod bằng Menu

3. Nếu bạn sử dụng Tool Mod Auto bằng việc chọn Resources.zip, vui lòng vào thư mục như sau: Tên File > [Files_Mod] > files, đến khi thấy thư mục Resources thì nén nó lại.

## Chú ý nếu bạn mod bằng Android

* Nếu Android 13 trở xuống

Hãy cóp nguyên folder Resources của File Mod vào folder game gốc (root folder game gốc = ZArchiver)

* Nếu Android 13 trở lên

Bạn sẽ không thể mod như Android cũ! Vui lòng:

1. Tìm Resources Android mới nhất gỡ lỗi không thể mod (tìm trên Youtube, và là Resources gốc)

2. Sau đó làm như Android 13 trở xuống!
